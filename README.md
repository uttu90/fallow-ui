# fallow-ui
Fallow-ui project

Start virtual env:
```
source ./bin/activate
```
Install dependencies:

```
pip install -r requirements.txt
```

Start your project

Convert qt designers ui to py

```
pyuic5 input.ui -o output.py
```

Run projects

```
fbs run
```
fbs tutorial: https://github.com/mherrmann/fbs-tutorial
